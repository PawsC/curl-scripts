This is my first somewhat decently sized project.<br>
It's mainly about overcoming RKN's whitelists with the help of publicly available VLESS VPN configs from www.github.com/igareck.<br>
I'll mainly try to focus on Termux compatability with the script since whitelists mainly affect me when I'm outside and Termux is my go-to option.<br>
In order to get everything going in just one command - copy, paste and execute the command below in the terminal:<br>
```
wget https://github.com/PawsC/Public-VLESS-configs-local-tester/raw/refs/heads/main/public_vless_configs_local_tester.sh
chmod +x public_vless_configs_local_tester.sh
./public_vless_configs_local_tester.sh
```
